Атрибуты тега script (async и defer)

У тега script есть два атрибута, async и defer, которые могут дать нам больше контроля над тем, как и когда файл загружаются и выполняются.

async

Скрипт загружается независимо от DOM дерева. Браузер строит DOM дерево без ожидания завершения загрузки скриптов. Скрипты выполняются Если скриптов несколько с атрибутом async, то выполнится тот скрипт, который загрузится первым, а первым обычно загружается файл скрипта меньший по размеру.

defer

Скрипт загружается в фоновом режиме и не блокирует контент. Браузер после завершения формирования дерева DOM выполняет скрипты. Если скриптов несколько с атрибутом defer, то скрипты выполняются в том порядке в каком они были записаны в HTML файле.

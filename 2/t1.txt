Почему код даёт именно такие результаты?

var a = 1, b = 1, c, d;

1)
c = ++a; alert(c);           // 2
Префиксная запись инкремента, сначала происходит инкрементирование (+1), а потом уже возврат измененного значения в переменную
c = 1+1; 

2)
d = b++; alert(d);           // 1
Постфиксная запись инкремента, сначала происходит возврат значения в переменную, а потом только инкрементирование (+1)
d = 1;

3)
c = (2+ ++a); alert(c);      // 5
Префиксная запись инкремента, сначала происходит инкрементирование (+1), а потом уже возврат измененного значения в переменную
c = (2+ (2+1)));

4)
d = (2+ b++); alert(d);      // 4
Постфиксная запись инкремента, сначала происходит возврат значения в переменную, а потом только инкрементирование (+1)
d = (2+ 2); 

5)
alert(a);                    // 3
3 - это 1 (a из инита) => 2 (a++ из пункта №1) => 3 (a++ из пункта №3)

6)
alert(b);                    // 3
3 - это 1 (b из инита ) => 2 (b++ из пункта №2) => 3 (b++ из пункта №4)
